# Outputs

This directory stores example artifacts generated by `examples/demo.py`.

It is intentionally tracked in the repository for now so that:

- the Markdown export format is visible on GitHub
- the JSON artifact structure is easy to inspect
- `artifact_index.json` can serve as a lightweight example catalog

Current subdirectories:

- `markdown/`: exported Markdown artifacts
- `json/`: exported JSON artifacts
- `artifact_index.json`: index of exported artifacts

If the project later moves toward cleaner production-style version control, a common next step would be:

- keep only a few curated sample artifacts
- ignore newly generated runtime outputs by default
